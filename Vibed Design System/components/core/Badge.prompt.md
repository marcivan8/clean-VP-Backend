A pill status badge with a glowing "LED" dot and mono-uppercase label — Vibed's signature instrument-panel indicator.

```jsx
<Badge tone="cyan">Live</Badge>
<Badge tone="rec">Rec</Badge>
<Badge tone="neutral" dot={false}>Draft</Badge>
```

Tones: cyan, violet, success, warning, danger, rec, neutral. Set `dot={false}` for a plain label pill.
