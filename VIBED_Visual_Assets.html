Usage guide

Comment utiliser ces assets

Screenshot individuel : Ouvre ce fichier dans Chrome, fais un clic droit sur chaque card → "Inspecter" → click droit sur l'élément → "Capture node screenshot". Tu obtiens l'image exacte à la bonne résolution.

Tailles exactes : Les dimensions affichées (ex: 1080×1920) correspondent aux dimensions de production. Pour l'affichage à l'écran, les cards sont mis à l'échelle — les proportions sont correctes.

Personnalisation : Remplace [ton_compte] par ton vrai handle. Remplace l'emoji 🎬 par ta vraie photo de profil dans les thumbnails YouTube.
